## Getting Started

### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/VladKoroliuk/todos-server
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
### Start server

1. Start server for development
    ```sh
   npm run dev
   ```
   Start server for production
    ```sh
   npm run server
   ```
